관심분야

  • 정보보안
    • 암호화(Encryption) : 데이터를 다른 사람이 읽지 못하도록 변환하는 기술
    • 취약점(Vulnerability) : 해커가 침입할 수 있는 시스템의 약한 부분
    • 네트워크 보안(Network Security) : 네트워크를 통해 발생하는 공격을 막는 기술
    • 악성 코드(Malware) : 시스템에 피해를 주는 바이러스, 랜섬웨어 등의 프로그램
  • 네트워크 기술
    • 통신 프로토콜(Protocol) : 컴퓨터 간 데이터를 주고받기 위한 규칙
    • 네트워크 속도(Network Speed) : 인터넷에서 데이터가 얼마나 빠르게 오고 가는지를 나타내는 것
    • IP 주소(IP Address) : 인터넷에서 각 기기를 구별하는 고유한 주소
    • 패킷(Packet) : 데이터를 작은 조각으로 나눠서 전송하는 단위
  • 프로그래밍
    • 알고리즘(Algorithm) : 문제를 해결하기 위한 순서대로 정리된 방법
    • 디버깅(Debugging) : 코드에서 오류(버그)를 찾아서 고치는 작업
    • 변수(Variable) : 값을 저장하고 변경할 수 있는 메모리 공간
